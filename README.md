# ruby-on-wheels
Simple car marketplace using Rails as API and Nuxt/VUE.js as frontend.

# Backend
 - Rails 6.1.4.1
 - API
 - Authentication via JWT
 - Testing using Rspec
 - Running on port 5000
# Frontend
 - Nuxt 2.15.7
 - Axios
 - NuxtJS/auth-next
 - Vuetify
 - Running on port 3000

# How to run
  1. cd backend
  2. bin/rails s
  3. cd ../backend
  4. yarn dev
  5. visit http://localhost:5000
  6. login:
    user: test@row.com
    pass: 123123123
